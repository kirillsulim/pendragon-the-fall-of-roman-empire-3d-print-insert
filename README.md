## About

These are 3D printable models for an insert for **Pendragon: The Fall of Roman Britain** by GMT Games.  
Each faction has its own box, designed to fit exactly the number of components needed for play — making it very easy to check for missing pieces.

A spare box is included to store blank tokens from sheets, as well as any extra components that don't fit in their designated boxes (for example, the 4 prosperity cubes won't fit in the prosperity box; you can store them in the start box or faction boxes instead).

## Known issues

The start token box may require a coin to open (coin not included — pun intended).
